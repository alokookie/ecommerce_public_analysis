# ecommerce_public_analysis
Repositori ini berisi analisis data dari E-Commerce Public Dataset dan visualiasi untuk menampilkan kategori produk paling sering dan paling jarang dibeli, rata-rata jumlah produk per pesanan, dan tren nilai transaksi per bulan.
# Dashboard Analisis E-Commerce

## Deskripsi
Dashboard ini dibuat untuk menganalisis perilaku pelanggan dari dataset E-Commerce Public Dataset.

## Fitur
- Menampilkan kategori produk paling sering dan paling jarang dibeli
- Menampilkan rata-rata jumlah produk per pesanan
- Menampilkan tren nilai transaksi per bulan

## Cara Menjalankan

1. Install dependencies:
pip install -r requirements.txt

2. Jalankan streamlit:
streamlit run dashboard/dashboard.py
